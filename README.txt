JONNYPLAYER - Proyecto Android (Kotlin, ExoPlayer)
------------------------------------------------
Rutas: Importar en Android Studio (File -> Open) la carpeta que contiene settings.gradle

Instrucciones rápidas:
1) Abre Android Studio -> File -> Open -> selecciona la carpeta JONNYPLAYER
2) Espera a que Gradle sincronice
3) Si quieres reproducir archivos locales: toca la carátula en la app y selecciona un archivo de audio
4) Para generar APK: Build -> Build Bundle(s) / APK(s) -> Build APK(s)

Notas:
- Este proyecto usa ExoPlayer. Asegúrate de tener conexión a Internet para descargar dependencias.
- Si quieres que cargue carátulas remotas, puedo añadir Glide y las líneas necesarias.
